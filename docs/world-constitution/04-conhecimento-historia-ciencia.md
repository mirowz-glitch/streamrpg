# 4. Conhecimento, História e Ciência

**Status: ✅ Fundacional.**

## O Conhecimento

Nenhum livro possui toda a verdade.

Nenhum pesquisador sabe tudo.

Nenhum mapa está completo.

Nenhum historiador é totalmente confiável.

Todo conhecimento deve possuir limitações.

## A História

Não existe narrador onisciente.

Toda informação veio de alguém.

Logo... toda informação pode estar errada.

## A Ciência

Os estudiosos pesquisam. Erram. Discutem. Mudam teorias. Publicam novos
livros. Descobrem espécies. Revisam mapas.

O conhecimento evolui junto com o jogo.
