# 5. Quests e Narrativa

**Status: ✅ Fundacional.**

## As Quests

Nenhuma quest existe apenas para gerar XP.

Toda quest deve responder pelo menos uma destas perguntas:

- O que aprendi sobre esse povo?
- O que descobri sobre esse lugar?
- Quem mudou?
- Que lembrança ficará comigo?

Se nenhuma resposta existir... a quest deve ser refeita.

## A Emoção

Jamais manipular emoção. Ela deve surgir naturalmente.

Nunca criar tragédias apenas para fazer o jogador chorar.

Pequenas histórias possuem mais força.

## O Humor

O mundo também sorri.

Existem piadas locais. Ditados engraçados. Superstições absurdas.
Discussões inúteis.

NPCs brincam. Crianças inventam histórias. Velhos exageram.
