# 8. Títulos e Ditados

**Status: ✅ Fundacional.**

## Os Títulos

Jamais criar títulos genéricos.

Não:

- Matador de Dragões
- Mestre Ferreiro
- Caçador Supremo

Preferir:

- Aquele que Plantou Amanhã
- Último Aprendiz de Bren
- Escutou o Último Sino
- Nunca Fechou a Porta
- Voltou Antes do Inverno

O título deve despertar curiosidade.

## Os Ditados

Todo Reino possui ditados próprios.

Toda profissão possui ditados próprios.

Toda família possui frases próprias.

Os ditados podem se contradizer. Assim como acontece no mundo real.
